# Meteorstrap-bootswatch

### A collection of themes by [Bootswatch](https://github.com/thomaspark/bootswatch), compatible with [tap:meteorstrap](http://github.com/tapevents/meteorstrap)

Currently Includes:

* amelia
* cerulean
* cosmo
* yeti
* united
* cyborg
* darkly
* flatly
* lumen
* readable
* simplex
* slate
* spacelab
* superhero
